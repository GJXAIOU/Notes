# 用户权限

MySQL 对于用户权限的限制可以划分为三个层级：

- 是否允许用户连接登录到 MySQL 服务器；
- 用户可以访问哪些数据库；
- 用户可以访问哪些表，同时可以定义对表的操作权限，select、insert、update…
- 用户可以对某表的某列进行操作；

## 一、保存用户权限
