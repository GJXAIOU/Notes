# idea导入第三方JAR包


## 方法/步骤

1.点击 File ->  Project Structure（或者快捷键 Ctrl + Alt + Shift + s）；

2.点击Project Structure -> “Modules”；

3.“在Dependencies” 标签界面下；

4.点击右边绿色的 “+”号；

5.选择第一个选项“JARs or directories...”；

6.选择相应的jar包，点“OK”，jar包添加成功；

7.点“OK”返回项目界面；

8.Alt+Enter 键快速添加import语句之后，项目就能编译；
